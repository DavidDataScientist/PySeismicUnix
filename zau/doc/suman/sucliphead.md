# sucliphead

SUCLIPHEAD - Clip header values

## Synopsis

```bash
sucliphead <stdin >stdout [optional parameters]
```

## Required Parameters

none

## Optional Parameters

key=cdp,...			header key word(s) to clip
min=0,...			minimum value to clip
max=ULONG_MAX,ULONG_MAX,...	maximum value to clip
