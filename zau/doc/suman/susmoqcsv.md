# susmoqcsv

SUSMOQCSV - Input Q-records, alter and spatially smooth, output Q-records. susmoqcsv [parameters].   (No traces in or out). Parameters:

## Synopsis

```bash
susmoqcsv =f,l,i start at f, increment by i, end at last value <= l.
```

## Notes

All input values are stored in double-precision (8 byte float).
This program does not care if the values are seconds or milliseconds
or feet or metres. To this program they are all just numbers.
