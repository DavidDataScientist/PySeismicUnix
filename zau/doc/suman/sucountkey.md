# sucountkey

SUCOUNTKEY - COUNT the number of unique values for a given KEYword. sucountkey < input.su key=[sx,gx,cdp,...]

## Synopsis

```bash
sucountkey < input.su key=[sx,gx,cdp,...]
```

## Optional Parameters

verbose=1		chatty, =0 just print keyword number

## Examples

```
suplane | sucountkey key=tracl,tracr,offset
```
