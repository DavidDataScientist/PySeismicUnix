# suzero

SUZERO -- zero-out (or set constant) data within a time window suzero itmax= < indata > outdata

## Synopsis

```bash
suzero itmax= < indata > outdata
```

## Required Parameters

itmax=		last time sample to zero out

## Optional Parameters

itmin=0		first time sample to zero out
value=0		value to set
