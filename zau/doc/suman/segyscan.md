# segyscan

SEGYSCAN -- SCANs SEGY file trace headers for min-max in  several possible formats. segyscan < segyfile

## Synopsis

```bash
segyscan < segyfile
```

## Notes

The SEGY file trace headers are scanned assuming short, ushort, int,
uint, float, and double and the results are printed as tables.
